h1. "AnZen-Lab (UEC, Tokyo)":http://www.az.inf.uec.ac.jp/  Thesis Styles(TeX+BibTeX)

LaTeX style formats, samples and utilities. Citing and referencing using BibTeX.
These file encoding is EUC-JP.

h2. Requirements

Makefile uses above software.

* platex
* bibtex
* dvipdfmx

So if you want to use "Make" command, you have to install there software before using.
